# Project Overview

This is a spec-driven console todo application built in Python with clean architecture.

## Purpose

The application allows users to manage their tasks through a command-line interface. It implements basic CRUD operations for tasks with the ability to mark tasks as complete or incomplete.

## Architecture

The application follows clean architecture principles:

- Models: Define the data structures
- Services: Handle business logic
- CLI: Handle user interface and interaction

## Technology Stack

- Python 3.8+
- Standard library only (no external dependencies required)
